```bash
ping 10.150.150.145
```
First we need to ping the machine 10.150.150.145 to see if the host is up.

![gambar1](img/ping1.png)

So, we can see the host is up! Now we can start our first step which is information gathering.

```bash
sudo nmap -sC -sV -vvv -Pn 10.150.150.145
```
nmap : Tool uses for network discovery.
-sC : Runs Nmap's standard set of built-in scripts for service enumeration
-sV : Probes open ports to identify service/application versions
-vvv : Very verbose output (level 3), Shows maximum detail about scan progress and findings
-Pn : Treats target as online without ICMP ping probes (bypasses ping sweeps)
